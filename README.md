🏧 ATM Banking System
A Python-based ATM Banking System with a web interface built using HTML, CSS, and JavaScript. This project simulates core ATM functionalities including user authentication, balance inquiry, deposits, withdrawals, and transaction history.
Features: The system includes secure login with account number and PIN, balance checking, deposit and withdrawal functionality with validation, and a transaction history viewer — all accessible through a clean web interface.
Tech Stack: The backend is built with Python while the frontend uses HTML, CSS, and JavaScript for the user interface.
How to Run: Clone the repository using git clone https://github.com/intabdul/ATM_Banking_System.git, install dependencies with pip install -r requirements.txt, run the app with python app.py, and open http://127.0.0.1:5000 in your browser.
Concepts Used: This project applies Object-Oriented Programming (OOP), backend logic with Python, frontend integration, and input validation with error handling.
